# ClearOffice
Install three office applications from ClearGroups in one click.

# RU:

# Возможности
- Установка 3-х программ по нажатию одной кнопки, без распаковки доп. файлов.
- Простой интерфейс

# Установка

Скачай ClearOffice.exe из [Releases] (https://github.com/ClearGroups/ClearOffice/releases) и запусти.
